index.html 的摘要
這是一個使用 HTML、CSS 和 JavaScript 建立的多語言單字匹配記憶遊戲。它的功能如下：

核心功能
遊戲機制：
玩家用中文和泰米爾語（坦米爾文）配對卡片
點擊卡片，從洗亂的牌堆中找到匹配的卡片對。
計時器會記錄完成時間，並在結束後顯示分數。

自訂選項：
字體選擇：9 種不同的英文手寫/顯示字體
文字大小：英文和中文字元大小分別有獨立的滑桿。
字數：每局遊戲選擇 6、12 或 21 對單字
自訂詞彙：請按以下格式新增您自己的字詞對：Chinese | Tamil | [phonetic]

輸入法：
語音輸入：語音辨識（支援中文）
鍵盤輸入：直接輸入文字並點擊提交按鈕
線上翻譯：如果內建字典中沒有所需的單詞，則回退到 MyMemory 翻譯 API。
內建內容：
100多個預先載入的詞組，涵蓋動物、食物、自然、交通工具、建築物、身體部位和服裝，並配有表情符號插圖。

技術亮點
響應式網格佈局（行動端 3 列，桌面端 4 列）
視覺回饋：元音高亮顯示，正確/錯誤匹配的動畫效果
點擊卡片即可進行文字轉語音朗讀。
單字查找的 Toast 通知
遊戲進程和卡牌補充的狀態管理
遊戲既支援預訂詞彙表，也支援使用者創建的自訂詞彙表，使其在語言學習方面具有靈活性。
