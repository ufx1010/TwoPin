LinguaBook 網頁功能分析
這是一個完整的雙語歌詞影音學習應用程式。

核心功能
1. 媒體播放與同步
支援音檔、影片、圖片播放
歌詞與媒體時間軸同步顯示
播放進度條拖曳控制
前後曲導航
2. 雙語歌詞編輯器
區塊模式：逐行編輯，帶時間標記 [mm:ss]
文本模式：純文字輸入，自動格式化
支援智能斷詞（中日文逐字、其他語言按詞）
時間戳快速設定小鍵盤
3. 語音合成與播放
自動語言偵測（英、中、日、韓、阿拉伯、坦米爾等多語言）
文字轉語音播放
語速調整（0.5x～1.5x）
可獨立控制原文/譯文聲音開關
4. 練習模式 🎤
語音辨識：錄音並自動辨識發音
容錯度調整（1-100%）：控制比對的嚴格程度
即時反饋：視覺化高亮顯示正確/錯誤部分
手動輸入模式：無麥克風也可打字練習
語言切換：練習原文或譯文
5. 打點同步 ⏱️
邊播放邊點擊字幕，自動記錄每行的時間點
適合沒有現成時間標記的歌詞
進階功能
6. 歌詞管理
新增/編輯/刪除歌曲
排序方式：自訂順序、A-Z、Z-A、最新/最舊
快速切換歌曲清單
拖拽排序（自訂模式）
7. 媒體存儲
IndexedDB 本地存儲音檔和圖片
支援多首歌曲獨立保存
分頁間自動同步資料
8. 匯入/匯出
匯出單曲備份（JSON 格式）
匯入備份恢復歌詞
匯出 LRC 歌詞檔（原文/譯文分開）
匯出完整影片（含字幕）
9. 語言模板
預裝 13 個語言示範模板：

🇬🇧 英文、🇹🇼 中文、🌐 英中混合
🇯🇵 日文、🌐 日中混合
🇰🇷 韓文、🇪🇸 西班牙文、🇫🇷 法文
🇸🇦 阿拉伯文、🇮🇳 坦米爾文、印地文
🇧🇷 葡萄牙文、🇷🇺 俄文、🇹🇼 台語
複製模板可轉換為可編輯歌曲
10. 字型與外觀客製化
字級調整（數值滑桿）
顏色選擇（預設色盤 + 自訂）
設定持久化儲存
11. 全螢幕與手勢控制
隱藏網址列全螢幕模式
圖片/影片手勢操作：
雙指捏合：縮放（0.3x～4x）
單指拖拽：移動（放大時）
雙擊：重置位置
字幕可浮動拖曳
12. 側邊錄製 📹
只錄製「媒體區 + 字幕」，不含按鈕/網址列
動態繪製字幕到錄製視訊
音訊從媒體播放器抽取
支援自訂字級和顏色
13. 狀態恢復與多分頁同步
自動儲存播放進度、語速、聲音/文字開關
頁面隱藏時存檔，返回時恢復
偵測其他分頁的更新，自動重新整理
防止舊分頁資料覆蓋新資料
14. 深色主題 UI
Tailwind CSS 深色設計
玻璃態效果（Glassmorphism）
流暢動畫轉場
行動裝置最佳化（響應式設計）
技術棧
前端框架：Vanilla JavaScript（無框架依賴）
樣式：Tailwind CSS
字體：Google Fonts（Outfit、Fraunces）
瀏覽器 APIs：
Web Audio API（音訊處理）
Web Speech API（文字轉語音 + 語音辨識）
IndexedDB（本地資料庫）
Canvas API（側邊錄製繪圖）
Fullscreen API（全螢幕模式）
使用情境
✅ 自學外語歌曲與發音
✅ 雙語內容製作與管理
✅ 語言練習與口語訓練
✅ 創意影音教學內容製作
